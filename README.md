# Study Hub

**Study Hub** is a modern collaborative platform designed to help students study, discuss ideas, and share academic resources in one place. The platform combines real-time communication, AI-assisted research, and a shared library of course materials to create a powerful learning environment.

## Features

### Authentication

* Username and password login
* Google (Gmail) authentication
* Secure password encryption
* Forgot password with email reset

### User Roles

**Admin**

* Full system control
* Manage users and uploaded content
* Moderate discussions
* Access admin dashboard

**Members**

* Participate in discussions
* Upload and access course modules
* Chat with other users

### Communication

* Real-time group discussions
* Private one-on-one messaging
* Message reactions
* Threaded replies
* Image and file sharing

### Course Module Library

* Upload and store study materials
* Organize files by course or subject
* Search and preview documents
* Download shared modules
* Track who uploaded each file

### AI Research Assistant

* Built-in AI assistant powered by ChatGPT 5.2
* Ask questions about study topics
* Summarize modules
* Explain complex concepts
* Assist with research

### Calls

* Voice calls
* Video calls
* Group study sessions

### Notifications

* Message notifications
* Replies to your messages
* New module uploads

### Interface

* Clean and responsive design
* Works on desktop, tablet, and mobile
* Light mode and dark mode
* Modern chat-style interface

## Technology Stack

Frontend

* React / Next.js

Backend

* Node.js with Express

Database

* PostgreSQL or MongoDB

Authentication

* JWT + Google OAuth

Real-Time Communication

* WebSockets / Socket.io

Video and Voice Calls

* WebRTC

File Storage

* Cloud storage

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/study-hub.git
```

2. Navigate into the project

```
cd study-hub
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

5. Open in browser

```
http://localhost:3000
```

## Project Goal

Study Hub aims to create a centralized space where students can collaborate, access study materials, and use AI to improve their understanding of course content.
